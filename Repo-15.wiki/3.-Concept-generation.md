# PROBLEM DEFINITION:
Design an automatic cup crusher machine, which is able to crush and shred the used cups, it should be portable and user friendly. Machine must not exceed the volume of 3 cubic feet and weight of 2Kg , should be able to crush and shred at least 10 cups per minute, should be liquid proof, budget to build should not exceed â‚¹ 2000, the waste from this process should directly enter a dustbin, should use minimal electricity and it should work on both AC and DC power(batteries).

 INPUTS | OUTPUTS
 ------ | -------
 Electricity | Heat
 Paper cups | Crushed cups
 Commands | Alerts

## LIST THE FUNCTIONS OF THE SYSTEM
1. It should hold and sense the cups.
2. It should count number of cups.
3. It should shred the crushed cups to smaller pieces.
4. It should alert on/off and completion of crushing.
5. It should direct waste to a dustbin.
